# Prompt: Implement Buttons (Bible-Compliant)

Task:
Create Bible-compliant button usage for a screen with:
- Primary action: Save
- Secondary action: Cancel
- Destructive action: Delete

Rules:
- Use system `Button` (SwiftUI) or `UIButton(type: .system)` (UIKit).
- Use roles: `.destructive`, `.cancel` where applicable.
- Use only system styles (e.g., `.borderedProminent`, `.bordered`, plain).
- No custom colors, no custom corner radius, no custom padding numbers.
- Accessibility: labels and hints as needed; icon-only must have labels.

Deliver:
- SwiftUI implementation and UIKit implementation
- A short compliance declaration.
